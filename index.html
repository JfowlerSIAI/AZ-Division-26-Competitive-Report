Methodology & Key Findings

The points formula

EventPoints = max(1, (N - Place + 1) * Strength(EventRating)); season total = cumulative sum. Last place always scores >=1 to encourage attendance.

N = number of fencers in the event; Place = finish (ties share a place → equal points; last place = 1, floored so attendance always counts). Event difficulty is the USA Fencing classification: within a letter a higher number is stronger (A2 > A1; A4 strongest). Cumulative totals reward consistent competition; PPE is shown for quality.
